# Sistema de Tickets de Servicio con IA

Este es un sistema de tickets de servicio accesible e innovador que utiliza IA para clasificar problemas y sugerir soluciones automáticamente. Implementa una interfaz moderna con soporte para **modo oscuro** y **accesibilidad WAVE**.

## Características:
- **Clasificación de tickets con IA**
- **Modo Oscuro/Claro**
- **Sugerencias automáticas de solución**
- **Accesibilidad para usuarios con discapacidades (WAVE)**
- **Diseño minimalista con neumorfismo**

## Requisitos del sistema:

### Frontend:
- HTML5
- CSS3
- JavaScript

### Backend (opcional para la IA):
- Python 3.x
- Flask

## Instalación y Configuración:

1. Clona el repositorio en tu máquina local:
    ```bash
    git clone https://github.com/tu-usuario/service-ticket-system.git
    ```

2. Navega al directorio del proyecto:
    ```bash
    cd service-ticket-system
    ```

3. Instala las dependencias del backend:
    ```bash
    pip install -r requirements.txt
    ```

4. Ejecuta el servidor:
    ```bash
    python server.py
    ```

5. Abre `index.html` en tu navegador para ver la interfaz de usuario.

## Dependencias:

El archivo `requirements.txt` incluye las dependencias necesarias:
```
Flask==2.0.1
```
